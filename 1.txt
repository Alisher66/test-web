Задачи на сообразительность
1 задача
Есть 90 человек. Каждый из этих 90 человек относится к одной из двух категорий. Одни на все вопросы говорят правду, другие — всегда врут. У каждого ровно 1 любимое блюдо из списка (макароны/пельмени/вареники). Каждому задают три вопроса:
Ваше любимое блюдо — макароны?
Ваше любимое блюдо — пельмени?
Ваше любимое блюдо — вареники?
Результаты получились такие: 
на первый вопрос утвердительно ответило 45 человек,
на второй — 35,
на третий — 30.
Сколько человек всегда говорят правду?


Решение:

x => всегда говорить правду 
y => всегда врут
всего 90 человек.

x+y = 90

у каждого ровно 1 любимое блюдо из 3-ех (макароны/пельмени/вареники)
задаётся 3 вопросы и 
    -у категория "х" на вопросы всегда отвечает правду, есть только один правильный ответ для каждого вопроса,
будем обозначать ответы от категории х => x1, x2, x3 (x1,x2,x3 это переменные)
    -у категории "y" на вопросы всегда врут, каждый человек из категории "y" на вопрос, если это и есть любимое блюдо отвечает нет, 
    на других двух ответ утвердительно да чтобы лгать, очевидно отвечает 2 раз утвердительно из этой категории, 
    будем обозначать ответы от категории y => 2*y1, 2*y2, 2*y3 (y1, y2, y3 это переменные)


x1+x2+x3 = x
y1+y2+y3 = y
x+y = 90
y = 90 -x
сумма всех ответов = 45+35+30 = 120

x1 + x2 + x3 + 2*y1 + 2*y2 + 2*y3 = 120
x1 + x2 + x3 + 2(y1 + y2 + y3) = 120
x + 2*y = 120
x + 2*(90 - x) = 120
x +180 -2x = 120
x = 60
Ответ: 60 человек всегда говорят правду
